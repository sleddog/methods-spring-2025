Count Mountain Peaks

Count Mountain Peaks will tell the user how many peaks are present in a given mountain range represented by a list of elevations

Introduction

This is a program that calculates the number of mountain peaks from a list of elevation values. The program takes elevation data as input and returns the number of peaks based on the given algorithm.

How to Use

To run the program, follow these steps:

  • Clone the repository to your local machine.
  • Run the Python script by passing a list of elevations.
  • Optionally, you can use the shell script to pass arguments easily.

Example Usage

Command to run the Python script directly:

python3 ./count_mountain_peaks/python/sethullamn/script.py 4500 7200 6800 8100 7900 9000 8500

Alternatively, use the shell script to pass the arguments:

./count_mountain_peaks/python/sethullman/run.sh 4500 7200 6800 8100 7900 9000 8500
