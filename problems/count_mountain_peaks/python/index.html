Count Mountain Peaks

Count Mountain Peaks will tell the user how many peaks are present in a given mountain range represented by a list of elevations

Introduction

This is a program that calculates the number of mountain peaks from a list of elevation values. The program takes elevation data as input and returns the number of peaks based on the given algorithm.

How to Start the Server

First, make sure you have Flask installed. If not, create a virtual environment and install Flask:

python3 -m venv venv
source venv/bin/activate
pip install flask

Then, start the server by running:

python3 app.py

The server will start locally on http://localhost:5000.

How to Make Requests

Open another terminal window and use curl to send POST requests to the server.

Example Requests:

  • curl -X POST http://localhost:5000/solve -H "Content-Type: application/json" -d '{"input": [1,3,2,4,1,5,2]}'
    Expected Response: 
    {
  "result": 3
}
  • curl -X POST http://localhost:5000/solve -H "Content-Type: application/json" -d '{"input": [5,2,6,8,2]}'
    Expected Response: 
    {
  "result": 1
}

Notes:

  • Make sure the server is running before sending requests.
  • The input must be a JSON object with a key called input that maps to a list of numbers.
  • If the input is invalid (e.g., not a list), the server will return an error message.

How to Use without the server

To run the program, follow these steps:

  • Clone the repository to your local machine.
  • Run the Python script by passing a list of elevations.
  • Optionally, you can use the shell script to pass arguments easily.

Example Usage

Command to run the Python script directly:

python3 ./count_mountain_peaks/python/sethullamn/script.py 4500 7200 6800 8100 7900 9000 8500

Alternatively, use the shell script to pass the arguments:

./count_mountain_peaks/python/sethullman/run.sh 4500 7200 6800 8100 7900 9000 8500
