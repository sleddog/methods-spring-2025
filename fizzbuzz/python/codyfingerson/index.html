BobCat FizzBuzz

This is a Python implementation of the FizzBuzz problem with a twist. The program replaces numbers based on the following rules:

  • If a number is divisible by 3 and 5, it prints BobCat.
  • If a number is divisible by 5, it prints Cat.
  • If a number is divisible by 3 or 5, it prints Bob.
  • Otherwise, it prints the number itself.

Usage

Python Script

Run the script directly from the command line:

python3 main.py 15

Example Output:


1
2
Bob
4
Cat
Bob
7
8
Bob
Cat
11
Bob
13

BobCat

Shell Script

Alternatively, you can run the shell script:

./run_fizzbuzz.sh 15

Ensure the script is executable by running:

chmod +x run_fizzbuzz.sh

Unit Tests

Unit tests are included to validate the program. To run them:

python3 -m unittest discover -s . -p "test_main.py"

Test coverage:

  • Basic number printing.
  • Correct replacement of numbers for "Bob", "Cat", and "BobCat".

Files

  • main.py - The main Python script implementing BobCat FizzBuzz.
  • run_fizzbuzz.sh - A shell script to run the Python script.
  • test_main.py - Contains unit tests for main.py.
  • run_tests.sh - A shell script to run all unit tests.
