Fizz Buzz

Description

This program contains a solution to the FizzBuzz problem, Write a function that prints numbers from 1 to n

  • For multiples of 3, print "Bob"
  • For multiples of 5, print "Cat".
  • for multiples of both 3 and 5, print "BobCat"

Running

To start the program run the run.sh script and when prompted enter a number.
